# Thomex — East African Marketplace

## Vision
A high-density, deal-driven ecommerce marketplace built for browsing abundance, urgency-driven purchasing, and mobile-first shopping.

## Design
- **Primary:** 190 100% 42% (turquoise #00B4D8) — CTAs, badges, active states
- **Secondary:** 214 97% 27% (french blue #023E8A) — headings, deep accents
- **Accent:** 201 100% 36% (blue-green #0096C7) — deal badges, urgency
- **Background:** 0 0% 100% (white) light / 239 60% 6% (deep twilight) dark
- **Foreground:** 239 94% 12% (near-black) light / 190 60% 96% (ice blue) dark
- **Muted:** 190 40% 96% (#F0F9FF) light / 239 40% 14% dark
- **Border:** 190 30% 88% light / 239 30% 20% dark
- **Success:** 142 71% 45% (#22C55E)
- **Warning:** 38 92% 50% (#F59E0B)

## Typography
- **Headings:** Sora, 700/800 — bold geometric
- **Body:** IBM Plex Sans, 400/500/600 — clean, readable at density
- **Data/Price:** IBM Plex Mono, 600 — tabular numbers for prices

## Features
- Homepage with hero banner, category pills, flash deals, product grid
- Product detail pages with related products
- Category browsing with filters
- Shopping cart with quantity controls
- Checkout flow with shipping info
- Order history page
- Search functionality
- User account dashboard
- Dark/light theme toggle
- PWA installable on Android, iOS, desktop
